# Cornell-Final-Lab
Uses the World Happiness Report (WHR) data set.  
The label is 'Life Ladder' or in other words 'Happiness'.  
This is an supervised learning, regression, and multi-class problem.
The features are Positive, Negative, LogGDP, Support, Life, Freedom, Generosity, Corruption.
This analyzes and bring insight into how the self-reported Happiness (Life Ladder) is dependent on a variety of different factors.
